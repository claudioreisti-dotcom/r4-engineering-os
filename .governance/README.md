# Governance

> Documentos **permanentes**. Não mudam de Sprint para Sprint. As missões mudam; a governança permanece.
>
> Qualquer alteração aqui exige uma Decision (ADR) aprovada pelo Human Governor.

Esta pasta define como o **próprio R4 Engineering OS** é desenvolvido: por um Human Governor assistido por uma workforce de especialistas de IA. É a constituição do projeto.

## Relação com `FOUNDATION.md`

`FOUNDATION.md` define os **valores de produto** (visão, identidade). A pasta `.governance/` define **como trabalhamos** para realizar esses valores. Onde houver sobreposição, o `FOUNDATION.md` é a fonte; aqui referencia-se, não se duplica.

## SSOT na própria governança

Cada documento é dono de exatamente um assunto. Nenhuma regra é repetida em dois lugares — os documentos se referenciam. Mudar uma regra muda um único arquivo.
