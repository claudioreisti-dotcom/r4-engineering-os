# Mission 0012 — Pilot Observation (suporte-r4)

## Status

In-progress (fase de observação; sem prazo fixo — encerra quando houver dados suficientes)

## Context

O suporte-r4 é o primeiro projeto real adotado pelo R4 EOS (bootstrap mergeado em
2026-06-24). Antes de qualquer nova evolução do modelo, o Governor quer **dados de uso real**.

## Mission

Manter um registro de observação do piloto e **congelar evoluções** de modelo/ontologia até
que o uso real justifique mudanças.

## Deliverable

- `docs/journal/0004-suporte-r4-pilot-observations.md` — registro vivo, com os eixos:
  assets que geram valor, assets obsoletos, informações atualizadas, lacunas naturais.

## Constraints (congelamento)

- Nenhuma evolução de modelo. Nenhuma missão de ontologia.
- Lacunas são registradas como candidatas, não aplicadas.

## Outcome

Registro aberto. Aguardando dados de uso real do suporte-r4.
