# Workforce Model

## Purpose

A Workforce Model define como o trabalho de engenharia é atribuído a Specialists.

## Human-Governed

A autoridade permanece humana.

## AI-Driven

A execução operacional é preferencialmente realizada por especialistas de IA.

## Initial Specialists

- Product Strategist Specialist
- Architecture Specialist
- Backend Specialist
- Frontend Specialist
- QA Specialist
- Documentation Specialist
- Knowledge Specialist
- Security Specialist
- DevOps Specialist
- Reviewer Specialist
