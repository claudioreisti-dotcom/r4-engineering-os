# Plugin System

## Purpose

O Plugin System permite que o R4 EOS seja estendido sem alterar o Kernel.

## Plugin Types

- Methodology Plugins
- Tool Adapters
- Capability Providers
- Compliance Plugins
- Domain Plugins
- Automation Plugins

## Examples

- Superpowers Plugin
- Elevate Plugin
- Claude Adapter
- Codex Adapter
- SBIS Compliance Plugin
- Healthcare Domain Plugin

## Principle

The Kernel is open for extension and closed for modification.
