# Journal 0004 — Observação do Piloto suporte-r4

> **Registro vivo.** Coleta dados de uso real do primeiro projeto adotado pelo R4 EOS,
> para informar futuras evoluções. **Esta fase é de observação, não de evolução.**

## Regra desta fase (congelamento)

- **Nenhuma evolução de modelo nem missão de ontologia** até haver dados de uso.
- Lacunas observadas são registradas como **candidatas**, nunca aplicadas de imediato.
- Uma evolução só será considerada quando um sinal de uso real a justificar.

## Baseline da adoção

- Projeto: **suporte-r4** (plataforma de suporte com pipeline de agentes IA).
- Adotado em **2026-06-24** via PR (squash) na `main` do suporte-r4.
- Kernel pinado: **1.2.0**. Total: **24 assets** — 5 components, 5 resources, 4 principles,
  3 specialists, 4 capabilities, 1 provider, 1 workflow (`pipeline`).
- Bootstrap: apenas artefatos declarativos; zero alteração de código/comportamento.
