# Journal

O Journal preserva descobertas, hipóteses e observações que ainda não viraram decisões, especificações ou arquitetura formal.

Regras:

- Journal não é fonte de verdade.
- Descobertas maduras devem ser promovidas para decisões, especificações ou arquitetura.
- Conversas estratégicas devem gerar entradas no Journal quando houver aprendizado relevante.
